最近一次剪藏

若系统关闭了 Chrome 通知、或剪藏时无提示，请看这里（由扩展写入 chrome.storage）。工具栏上的扩展图标角标 OK / ! 也表示最近一次成败。 


    

    

      使用 File System Access API 选择文章 .md 落盘目录（例如
      obsidian-wiki/raw/general/wechat-clips）。句柄保存在 IndexedDB，刷新后无需重选。
    

    

      
      
    

    


    
Obsidian 库根（可选 · 用于解耦）

    

      若希望文章目录与图片目录互不包含（例如文章在
      raw/general/wechat-clips、图片在 raw/general/_assets），请先选整个
      Obsidian 库的根文件夹（能同时包含上述两者的祖先）。扩展会用库根对文章、图片路径做
      resolve，在 Markdown 里写入 ../… 相对链接。

      未绑库根时：图片只能选在「文章目录」之下（或沿用每篇 <base>_assets/）。
    

    

      
      
    

    


    
图片保存目录

    

      已绑库根：选图时从库根打开选择器，可选与文章目录为兄弟文件夹，实现解耦。

      未绑库根：从文章目录打开选择器，图片须在文章根之下；或选「与文章同目录」；否则用每篇
      <base>_assets/。
